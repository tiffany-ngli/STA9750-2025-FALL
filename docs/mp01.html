📈 US #1 Films That Didn’t Debut at the Top

Show the code 
# 9. How many Films reached Number 1 in the US but did not originally debut there? 
# That is, find films that first appeared on the Top 10 chart at, e.g., Number 4 but 
# then become more popular and eventually hit Number 1 What is the most recent film to pull this off?

COUNTRY_TOP_10 |>
  filter(country_name %in% "United States") |>
  filter(category %in% "Films") |>
  group_by(show_title, week) |>
  summarize(weekly_rank) |>
  group_by(show_title) |>
  mutate(debut_rank = first(weekly_rank)) |>
  mutate(growth = if_else(debut_rank > weekly_rank & weekly_rank == 1, TRUE, FALSE)) |>
  filter(growth == TRUE) |>  
  distinct(show_title, .keep_all = TRUE) |>
  format_titles() |>
  datatable(options=list(searching=FALSE, info=FALSE))
